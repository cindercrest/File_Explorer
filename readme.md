# File Explorer in C

A simple command-line file explorer written in C.

## Features
- List files and directories
- Read file contents
- Write/append to files

## Concepts Used
- File I/O (`fopen`, `fgets`, `fputs`)
- Directory handling (`dirent.h`)
- Input handling in C
- Buffer management

## How to Run

```bash
gcc main.c -o main
./main
